# 使い方
1. Mac AppStoreからxcodeをインストール
2. reboot
3. リカバリーモード(command + R 起動)でそこのターミナルから csrutil disable 実行
4. 自由に shellscript を編集します
5. 順番通りに実行します
6. 楽しみましょう
